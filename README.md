# Conversal Dashboard — Franchise Mockup

Interactive dashboard mockup for franchise owners, built as a static HTML page with Chart.js.

## What is this?

A visual concept of how Conversal can provide franchise owners with real-time marketing performance insights — leads, cost per lead, campaign performance, and organic SEO data — per location.

Built for the Belisol franchise pitch but designed to be brand-agnostic and reusable for any multi-location business.

## Features

- **KPI scorecards** — leads, CPL, ad spend, conversions, pipeline value
- **Per-location breakdown** — performance comparison across showrooms
- **Channel analysis** — Meta vs Google vs Organic vs Direct
- **Campaign detail table** — full transparency per campaign per location
- **Organic SEO tracking** — showroom page traffic and top keywords
- **Dark mode support** — automatic, follows system preference
- **Responsive** — works on desktop and tablet
- **Interactive filters** — vestiging and period selectors (UI-only in mockup)

## Tech stack

- Vanilla HTML/CSS/JS (no build step required)
- [Chart.js 4.4.1](https://www.chartjs.org/) via CDN
- [DM Sans](https://fonts.google.com/specimen/DM+Sans) via Google Fonts

## Usage

Open `index.html` in any browser. No server required.

For GitHub Pages: enable Pages on the repo and it will deploy automatically.

## Data

All data is fictional / illustrative. No real client data is included.
